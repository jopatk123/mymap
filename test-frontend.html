🗺️ 初始显示设置功能测试

🔧 功能概述

本测试页面用于验证地图初始显示设置功能是否正常工作，包括：

  • ✅ 后端 API 接口（获取、更新、重置设置）
  • ⏳ 前端设置界面和交互
  • ⏳ 地图初始化应用设置

📋 测试步骤

  1. 访问管理后台： http://localhost:3000/admin/files
  2. 查找"初始显示设置"按钮： 应该在文件管理页面的头部区域
  3. 点击按钮打开设置对话框
  4. 测试设置功能：
    • 启用/禁用初始显示设置
    • 设置经纬度（建议测试：116.4074, 39.9042 - 北京天安门）
    • 设置缩放级别（建议测试：10-15之间）
    • 保存设置
  5. 验证设置效果：
    • 访问地图页面: http://localhost:3000
    • 检查地图是否按设置的位置和缩放级别初始化
